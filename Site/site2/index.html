Malware

Tipos mais conhecidos

Spyware

Malware 1

Este malware é projetado para rastrear e espionar o usuário. O spyware inclui rastreadores de atividade, coleta de toque de tela e captura de dados. Para combater as medidas de segurança, o spyware modifica as configurações de segurança. Muitas vezes se junta ao software legítimo ou a cavalos de Troia.

Caso

Adware

Malware 2

É o software com suporte a anúncios e é projetado para fornecer anúncios automaticamente. O adware é frequentemente instalado com alumas versões do software. Alguns tipos de adware são projetados somente para fornecer anúncios, mas também é comum que o adware venha com spyware.

Caso
Malware 3

Bot

Um bot é o malware projetado para executar automaticamente a ação, geralmente online. Enquanto a maioria dos bots é inofensiva, uma utilização progressiva de bots mal-intencionados são os botnets. Vários computadores estão infectados com bots que são programados para esperar calmamente os comandos fornecidos pelo invasor.

Ransomware

Malware 4

Este malware é projetado para manter preso um sistema de computador, ou os dados incluídos nele, até que o pagamento seja feito. O rasomware normalmente funciona criptografando os dados no computador com uma chave desconhecida ao usuário. Outras versões do rasomware podem lançar mão das vulnerabilidades de sistemas específicos para bloquear o sistema. O rasomware é espelhado por um arquivo baixado ou alguma vulnerabilidade de software.

Casos

Scareware

Malware 5

Este é um malware projetado para persuadir o usuário a executar uma ação específica com base no medo. O scareware simula janelas pop-up que se parecem com as janelas de diálogo do S.O. Essas janelas transmitem mensagens que afirmam que o sistema está em risco ou precisa da execução de um programa para voltar ao normal. Se o usuário concordar em executar o programa mencionado para limpeza, seu sistema será infectado com o malware.

Caso


Rootkit

Malware 6

É um malware projetado para modificar o S.O. e criar um backdoor. Os invasores usam o backdoor para acessar o computador remotamente. A maioria dos rootkits utiliza as vulnerabilidades do software para escalonar privilégios e modificar arquivos de sistema. Também é comum os rootkits modificarem a computação forense do sistema e as ferramentas de monitoramento, o que os torna muito mais difíceis de serem detectados. Muitas vezes, um computador infectado por um rootkit deve ser apagado e reinstalado.

Casos
Malware 7

Worms

Worms são códigos maliciosos que se replicam ao explorar, de forma independente, vulnerabilidades em redes. Eles normalmente deixam a rede mais lenta. Enquanto um vírus requer um programa host para a execução, os worms podem ser executados de forma autônoma. Exceto pela infecção inicial, eles não necessitam mais da participação do usuário. Após infectar um host, um worm pode ser transmitido muito rapidamente pela rede. Todos eles habilitam uma vulnerabilidade, uma maneira de se propagar, e todos contêm uma carga.



Man-in-the-middle(MitM)

Malware 8

O MitM permite que o invasor tenha o controle sobre um dispositivo sem conhecimento do usuário. Com esse nível de acesso, o invasor pode interceptar e capturar informações do usuário antes de transmiti-las ao seu destino desejado. Esses ataques são amplamente utilizados para roubar informações financeiras. Muitos malwares e técnicas existem para fornecer aos invasores recursos de MitM.

Caso
Malware 9

Man-in-the-Mobile(MitMo)

Uma variação do Man-in-the-middle, MitMo é um tipo de ataque usado para assumir o controle de um dispositivo móvel. Quando infectado, o dispositivo móvel pode ser instruído a extrair as informações confidenciais do usuário e enviá-las para os invasores. ZeuS, um exemplo de exploit com capacidade de MitMo, permite que os invasores capturem silenciosamente as mensagens de SMS de verificação de 2 passos enviadas para os usuários.
